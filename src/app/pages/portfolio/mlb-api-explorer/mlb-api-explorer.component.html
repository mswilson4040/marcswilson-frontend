REST API based off of Baseball Databank

Teams Endpoint
The Teams endpoint is capable of returning teams from any given year and team information for any given team
Players Endpoint
The Players endpoint is capable of returning players from a given year and detailed information of a given player
Ballparks Endpoint
The Ballparks enpoint is capable of returning all ballparks or a specific ballpark by calling /ballparks/:key
