MLB Stats REST API Explorer

REST API based off of Baseball Databank and MLB Advanced Media API

The teams endpoint takes in two arguments: Year and Team ID.
Example: /api/mlbstats/years/:yearID/teams/:teamID
/api/mlbstats/years/1977/teams/SEA

{{team.name}}

The players endpoint takes in one argument: Player ID.
Example: /api/mlbstats/players/:playerID
/api/mlbstats/players/id/griffke02

{{player.personal[0].fullName}}

The ballparks endpoint takes in one optional argument: Player ID.
Example: /api/mlbstats/players/:playerID
/api/mlbstats/ballparks
or
/api/mlbstats/ballparks/:key

The boxscores endpoint takes in three arguments: Year, Month, and Day.
Example: /api/mlbstats/boxscores/2017/7/27
/api/mlbstats/boxscores/:year/:month/:day
