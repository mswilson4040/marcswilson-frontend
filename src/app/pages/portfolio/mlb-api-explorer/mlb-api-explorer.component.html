MLB Stats REST API Explorer

REST API based off of Baseball Databank and MLB Advanced Media API

The teams endpoint takes in two arguments: Year and Team ID. The return is an array of type Team. 

/api/mlbstats/years/:yearID/teams/:teamID
/api/mlbstats/years/1977/teams/SEA
Year {{selectedTeamYear}}
{{team.name}}

The players endpoint takes in one argument: Player ID. The return is an array of type Players. 

/api/mlbstats/players/id/:playerID
/api/mlbstats/players/id/griffke02
Year {{selectedPlayersYear}}
{{player.personal[0].fullName}}

The ballparks endpoint takes in one optional argument: Player ID. The return is an array of type Ballpark 

/api/mlbstats/ballparks
/api/mlbstats/ballparks/:key

The boxscores endpoint takes in three arguments: Year, Month, and Day. The return is an array of type Boxscore 

/api/mlbstats/boxscores/2017/7/27
/api/mlbstats/boxscores/:year/:month/:day
